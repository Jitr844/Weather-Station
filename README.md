🌦️ Weather Station

A simple and responsive Weather Station web application built using HTML, Bootstrap, and JavaScript. The application fetches real-time weather data from a Weather API and displays current weather information for any city.

 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature
- ☁️ Weather condition with icon
- 📱 Responsive Bootstrap interface
- ⚡ Real-time API data
- ❌ Handles invalid city names

🛠️ Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- OpenWeatherMap API




⚙️ Getting Started

1. Clone the repository
